## https://github.com/JuliusBrussee/caveman

Here's the summary for your developer knowledge base:

---

## 1. Title
**Caveman — Token-Efficient "Caveman Speak" Skill for AI Coding Agents**

## 2. Category
**Tools**

## 3. Summary
Caveman is a skill/plugin for AI coding agents (Claude Code, Copilot, Cursor, and 40+ others) that compresses LLM output into ultra-concise "caveman speak," cutting ~75% of output tokens while preserving full technical accuracy. It offers multiple intensity levels — including a Classical Chinese mode — and companion tools for terse commits, PR reviews, and memory file compression.

## 4. Key Takeaways
- **Significant token savings**: Benchmarked 22–87% reduction in output tokens (avg 65%), lowering cost and latency
- **Accuracy can improve with brevity**: Cites research showing constrained brief responses improved accuracy by 26 percentage points on certain benchmarks
- **Multiple modes**: Lite, Full, Ultra, and 文言文 (Classical Chinese) levels let you dial in the compression
- **Companion skills included**: `caveman-commit` (terse commit messages), `caveman-review` (one-line PR feedback), `caveman-compress` (rewrite memory files to save ~45% input tokens)
- **Preserves what matters**: Code blocks, URLs, and technical content remain untouched — only prose is compressed
- **Trivial install**: Single command (`npx skills add JuliusBrussee/caveman`) works across 40+ agent platforms
