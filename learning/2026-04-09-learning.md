## https://github.com/memvid/memvid

TITLE: Memvid
LINK: https://github.com/memvid/memvid
CATEGORY: tools
SUMMARY: Memvid is a single-file memory layer for AI agents that replaces complex RAG pipelines and vector databases. It compresses embeddings into one portable file using video-style encoding, handling millions of text chunks with near-instant semantic search — no servers or databases needed.
KEY TAKEAWAYS:
- Replaces costly vector databases with a single portable `.mv2` file — no ChromaDB, SQLite, or external servers required
- Blazing-fast semantic search with sub-5ms local memory access and predictive caching
- Fully portable, version-controlled, and resilient — files can be git committed, scp'd, or shared like any other file
- Recently rewritten from Python to Rust for 10-100x performance improvements
- Benchmarks show +15% SOTA on LoCoMo, +76% multi-hop reasoning vs. industry average, and ultra-low latency at scale
- 100% open-source under Apache 2.0 license
TAGS: ai-memory, rag, embeddings, semantic-search, vector-database-alternative, ai-agents, rust, open-source, long-term-memory

**TITLE:** Scrapy

**LINK:** https://github.com/scrapy/scrapy

**CATEGORY:** tools

**SUMMARY:** Scrapy is a fast, high-level web crawling and scraping framework for Python used to extract structured data from websites. It is cross-platform, maintained by Zyte (formerly Scrapinghub) and a large open-source community, and scales from simple scraping tasks to crawling millions of pages.

**KEY TAKEAWAYS:**
- Define a single Spider to start crawling and extracting structured data from any website with ease
- Scales to millions of pages with built-in support for concurrency, throttling, and distributed crawling
- Export scraped data directly to JSON, CSV, or XML formats out of the box
- Cross-platform framework requiring Python 3.10+, installable via `pip install scrapy`
- Mature, battle-tested project with a large ecosystem of middleware, extensions, and integrations (e.g., Scrapy Cloud, Splash for JS rendering)
- Built-in support for handling requests, following links, managing cookies/sessions, and respecting robots.txt

**TAGS:** Python, Scrapy, WebScraping, Data, OpenSource, Crawling, ETL, DataExtraction
