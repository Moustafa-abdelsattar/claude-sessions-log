**TITLE:** Agent Skills

**LINK:** https://github.com/addyosmani/agent-skills

**CATEGORY:** tools

**SUMMARY:** A collection of 20 production-grade engineering workflow skills for AI coding agents, created by Addy Osmani. It enforces disciplined software development practices — from spec writing through deployment — preventing AI agents from taking shortcuts that skip tests, security reviews, and other quality gates. Compatible with Claude Code, Cursor, Gemini CLI, Windsurf, GitHub Copilot, and more.

**KEY TAKEAWAYS:**
- Provides 7 command-driven entry points (`/spec`, `/plan`, `/build`, `/test`, `/review`, `/code-simplify`, `/ship`) covering the full development lifecycle
- Includes 20 interconnected skills organized by phase: Define, Plan, Build, Verify, Review, and Ship
- Each skill contains anti-rationalization patterns — documenting excuses agents use to skip steps, with rebuttals to enforce rigor
- Ships with reference checklists (security, performance, accessibility, testing) and agent personas (code-reviewer, test-engineer, security-auditor)
- Platform-agnostic: skills are plain Markdown files that work with any agent system supporting system prompts
- Embeds best practices from "Software Engineering at Google" including Hyrum's Law, trunk-based development, and feature flags
