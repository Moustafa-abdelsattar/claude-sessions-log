**1. Title:** Opik-OpenClaw — Observability Plugin for OpenClaw Agents

**2. Category:** Tools

**3. Summary:**
Opik-OpenClaw is an official plugin that integrates the Opik observability platform with OpenClaw, exporting agent traces for monitoring and debugging. It captures LLM request/response pairs, sub-agent communications, tool executions, usage metrics, and cost data. The plugin runs within the OpenClaw Gateway process and supports both local and remote deployments.

**4. Key Takeaways:**
- Provides end-to-end tracing of agentic workflows: LLM calls, tool use, sub-agent interactions, and errors
- Includes a CLI with commands for installation, configuration, and status checking, plus an interactive setup wizard
- Supports flexible configuration via API credentials, project naming, and environment variable fallbacks
- Tracks token usage and cost data for optimizing LLM spend
- Built with TypeScript/Node.js (requires Node 22.12.0+)
